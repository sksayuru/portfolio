Sayuru Karunathilaka, Isuru Karunathilaka (Development)

The website for FollowUp was redesigned through a contest on 99designs. The platform, which helps users manage and track business follow-ups with potential clients and leads, was given a modern, user-friendly interface. The design focused on clear call-to-action buttons, intuitive workflows, and a professional layout to enhance the user experience. A fresh, responsive design was implemented to improve both functionality and visual appeal.
